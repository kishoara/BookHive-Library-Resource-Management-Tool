# BookHive-Library-Resource-Management-Tool

A simple and beginner-friendly Library Management System written in C, developed as part of the Programming Language I (C) Lab. This project allows users to add books, display books, issue/borrow books, and return books using file handling.This is a simple **Library Management System** built in **C** as part of my **Programming Language I (C) Lab** assignment.  
The program allows users to add books, view books, issue books, and return books using basic file handling.
